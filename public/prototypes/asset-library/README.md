# Asset Library 素材

把文件按下面的**确切路径和文件名**放进对应文件夹(页面里就是按这些路径引用的)。
扩展名要一致——如果你的文件是别的格式(比如图片是 png/webp、头像是 jpg),
要么自己转/改名成下面的名字,要么告诉我你的实际文件名,我来改代码里的引用。

## 需要的 7 个文件

| 类型 | 放这里 |
|---|---|
| 图片 1 | `images/img-01.jpg` |
| 图片 2 | `images/img-02.jpg` |
| 图片 3 | `images/img-03.jpg` |
| 视频 | `videos/video-01.mp4` |
| 头像 | `avatars/avatar-01.png` |
| 音频 | `audio/audio-01.mp3` |
| PDF | `pdf/doc-01.pdf` |

放好后刷新 `http://localhost:3000/prototypes/asset-library` 就能看到。
